void* bounded_queue_pop ( bounded_queue *  queue)

Removes an element from the bounded_queue. This will block until there's an element to be removed from the queue.

The bounded_queue must have been created using bounded_queue_init, and must not have been destroyed. Failing to do so results will have an undefined behavior.

Parameters
queueThe queue the element will be removed from.
Returns
The element that was removed from the queue.
