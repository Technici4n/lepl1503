mpz_t* factorize ( const mpz_t  n,
int *  factors_length 
)

Factorize a single number using Pollard's rho algorithm.

This function computes every prime divider of n, excluding n itself. The resulting primes are stored in the returned malloc-allocated array, in increasing order. The length of this array is stored in factors_length. The caller should ensure the memory is freed when it's done with it. This includes the GMP numbers and the array itself.

Parameters
nThe number to factorize.
factors_lengthWhere to store the number of factors.
Returns
The prime factors of n (but not n itself), malloc-allocated.
